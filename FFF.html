About the project

This project was made while taking part in a game jam. It is a game about dungeon crawling where encounters could be solved with fighting, fleeing or flirting. Three different classes give bonuses to different actions. You can find the game here!


My role

I created a dungeon map generating system. Each time the map was created, it would be different. The generation was modular, where I created prefabs for each possible piece and then used colliders to determine if a generated maze piece was allowed to exist or not, eliminating instant deadends. I also created the movement system for the player. It was limited to 90 degree turns and forwards movement to simulate retro adventure game feel.
