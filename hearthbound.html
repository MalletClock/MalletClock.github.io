About the project

This project was made while taking part in a game jam. It is a game about smithing items that customers want using a dice system to get the parameters right. You can find the game here!


My role

I created the dice system utilizing interfaces so each die could use the same die script. I also created the 3D dice that the player can pick up and throw. Some dice will cause the parameters to go up or down and the used dice are taken out of the players pool of available dice. For the dice I wanted to create a pixelated look. To achieve this, a shader was used to pixelate the 3D visuals of the objects.
